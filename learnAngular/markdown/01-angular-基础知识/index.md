# Angular
Angular版本说明：
- AngularJS --> v1.x
- Angular   --> v2.x及其以上版本
- 编写该文档的时候使用的 Angular v9.x 最新

## Angular CLI    

1. 安装脚手架
```
npm install -g @angular/cli
```

2. 创建工作空间和初始应用
```
ng new my-app[项目名称]
```

3. 运行应用
```
cd my-app[项目名称]
ng serve --open
```
